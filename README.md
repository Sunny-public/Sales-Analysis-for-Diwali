# Sales-Analysis-for-Diwali
This project focuses on analyzing Diwali sales data using Python to uncover customer purchasing behavior and provide actionable business insights. The goal is to help businesses improve customer experience and increase sales during festive seasons.

🚀 Key Objectives
Understand customer demographics (age, gender, marital status, etc.)
Identify top-performing product categories
Analyze purchasing patterns across different states
Provide insights to improve marketing strategies

🛠️ Tools & Technologies
Python 🐍
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

📈 Key Insights
Married women aged 26–35 are the highest buyers
Certain states contribute the most to overall sales
Specific product categories dominate festive purchases
Targeted marketing can significantly boost revenue

📂 Dataset
The dataset contains information about:
Customer demographics
Product categories
Purchase amount
Location details
